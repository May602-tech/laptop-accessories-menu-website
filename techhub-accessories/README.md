# TechHub Accessories

A light-blue and white business contact landing page for TechHub Accessories.

## Features

- Pure HTML, CSS, and JavaScript
- No Tailwind CSS
- Right-side rectangular navigation menu
- Accessories image slider with loop, buttons, auto-slide, and drag support
- Glass blur / matte glass design
- Gradient buttons with fill hover effect
- Contact business card section
